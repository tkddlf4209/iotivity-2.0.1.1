If downloading GTest or ArduinoUnit is not successful, then please do the follwoing procedures

For GTest Linux:
1. Download gtest from https://googletest.googlecode.com/files/gtest-1.7.0.zip
2. Extract it
3. Copy gtest-1.7.0/src/* to extlibs/gtest-1.7.0/src/
4. Copy gtest-1.7.0/include/* to extlibs/gtest-1.7.0/include
5. Provide permissions if necessary using chmod 777 extlibs/gtest-1.7.0/src/* and chmod 777 -R extlibs/gtest-1.7.0/include/*
6. Remove unnecessary files
